# C-rculoDeFogo
Círculo de Fogo é um sistema preditivo baseado em IA para estimar a velocidade de propagação de incêndios florestais. Integrando dados climáticos, ambientais e de cobertura do solo, o projeto visa melhorar a precisão das previsões, superando modelos tradicionais e otimizando respostas a incêndios.
